### Getting Started Next.js, Server Components, PostgresDB without ORM

```bash
# Run default migration (only if you db is empty)
npm run migration -- 20240101000001_start_up_migration/migration.sql

# Seed DB
npm run seed

# Install deps
pnpm install
# or
yarn install
# or
npm install

# Run project
pnpm dev
# or
yarn dev
# or
npm run dev
```
